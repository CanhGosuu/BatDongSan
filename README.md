# BatDongSan

## How to test Server

    git clone https://github.com/CanhGosuu/BatDongSan.git
    npm install

---

## usage

1.  Open Postman (https://www.getpostman.com/apps)
2.  Trong Headers:
    > key= "Authorization"
    > value = Token sau khi login
3.  Nếu dùng POST, trong body sử dụng:
    > x-www-form-urlencoded
